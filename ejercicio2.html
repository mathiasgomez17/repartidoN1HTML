HTML

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos, como textos, imágenes, videos y enlaces. Utiliza un sistema de etiquetas (tags) para organizar elementos y crear la estructura básica que los navegadores interpretan para mostrar el sitio.

Caracteristicas del HTML

Es un lenguaje de mercado, que se basa en elementos y etiquetas, estos mismos se organizan de manera jerarquica, es estatico, compatible y estandar, etc.

Historia del HTML

El HTML (HyperText Markup Language) fue creado por Tim Berners-Lee en el CERN en 1989-1991 para facilitar el intercambio de documentos científicos.

HTML:5

La versión actual (HTML5) es el estándar moderno, enfocado en la semántica (etiquetas significativas como article, nav, header) y la compatibilidad multimedia.
